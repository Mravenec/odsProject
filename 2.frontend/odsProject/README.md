# Project ODS

Proyecto React con estructura profesional y axios configurado para consumo de APIs.

## Estructura del Proyecto

```
src/
├── components/     # Componentes reutilizables
├── pages/         # Páginas principales
├── hooks/         # Custom hooks
├── services/      # Servicios y configuración de APIs
├── utils/         # Utilidades y constantes
├── assets/        # Imágenes y recursos estáticos
├── styles/        # Archivos CSS
├── contexts/      # Contextos de React
├── App.jsx        # Componente principal
└── main.jsx       # Punto de entrada
```

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Características

- ✅ React 18 con Vite
- ✅ Axios configurado para APIs
- ✅ Custom hooks para consumo de datos
- ✅ Estructura de carpetas profesional
- ✅ CSS modularizado
- ✅ Configuración de ESLint
